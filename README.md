Language
--------
Add the class `language` to the code block to enable language translation.

```html
<div data-es="Hola" data-en="Hello">
  <!-- This won't work -->
</code>
```

```html
<div class="language" data-es="Hola" data-en="Hello">
  <!-- This will work -->
</code>
```
